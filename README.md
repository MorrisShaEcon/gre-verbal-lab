# GRE Verbal Lab

GRE Verbal Lab is a local-first, adaptive learning app for GRE vocabulary,
synonym and antonym relationships, contextual usage, and mistake-driven review.

The project begins as a personal study tool and is designed to become a useful,
privacy-conscious open-source learning system.

## Product principles

- Track mastery at the **word-sense** level, not only the word level.
- Connect vocabulary to the questions and contexts in which it appears.
- Turn each mistake into a targeted future review task.
- Prefer active recall and transfer over passive recognition.
- Keep personal study data local by default.
- Treat AI-generated lexical relationships as suggestions until confirmed.

The public repository will not contain proprietary GRE questions, commercial
word lists, personal mistake records, or copyrighted preparation PDFs. Users may
import materials for private local study where permitted.
