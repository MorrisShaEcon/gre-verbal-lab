# Roadmap

## v0.1 — Product foundation

- Product requirements
- User flows and interactive wireframe
- Data model
- Architecture decision
- Versioning and open-source repository rules

## v0.2 — Vocabulary learning loop

- Import a user-owned vocabulary spreadsheet
- Word and word-sense library
- Active-recall review cards
- Confidence and response-time capture
- Review scheduling
- Local data backup and restore

## v0.3 — Lexical relationships

- Synonym, antonym, contrast, and confusable relationships
- Sense-specific relationship confirmation
- Word-family and contrast drills
- Relationship graph navigation

## v0.4 — Mistake notebook

- Manual question and context entry
- Attempt history
- Structured mistake-reason tags
- Automatic reinforcement tasks
- Original-question replay and transfer practice

## v0.5 — Adaptive engine

- Multi-dimensional mastery scores
- Exam-date-aware daily queue
- Lapse and interference weighting
- Weekly weakness report
- Explainable scheduling decisions

## v0.6 — AI assistance

- Suggested explanations and lexical relationships
- Similar-practice generation
- Human confirmation workflow
- Provenance and uncertainty display

## v1.0 — Public release

- Stable local-first PWA
- Accessibility review
- Import/export documentation
- Privacy and copyright review
- Contributor documentation
- Optional, separately designed cloud sync
