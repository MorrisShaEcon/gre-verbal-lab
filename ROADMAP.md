# Roadmap

## v0.1 — Product foundation · complete

- Requirements, architecture, data model, wireframe, and repository safeguards

## v1.0 — Import-driven prototype · frozen

- Demonstrated local import, basic recall, mistakes, and backup
- Reclassified as a prototype after direct usability testing

## v2.0 alpha — Database-first rebuild · complete

- [x] Private 2,535-word versioned catalog pipeline
- [x] Open pronunciation and lexical enrichment
- [x] Local-material occurrence evidence without source-text retention
- [x] Stable 70/30 stratified daily sampling
- [x] Pronunciation, lexical relations, context, and evidence on cards
- [x] Catalog/profile/event storage separation and v1 migration
- [x] Branded icon and private/public standalone separation
- Editorial completion moved to the v2.1 content-quality gate

## v2.1 alpha — Quiz-first learning and content quality · in progress

- [x] Four-choice definition questions with three formally validated distractors
- [x] System-inferred review result; no manual mastery rating in the core flow
- [x] Same-session replay after a wrong answer
- [x] Openly licensed human audio with per-file attribution
- [x] Source, rights, target-visibility, and sense-alignment queue gate
- [x] Canonical sense de-duplication, trusted-IPA/POS gates, and complete review
      of the former 315-sense formal pool
- [x] Per-kind source-checked synonym/antonym evidence without fabricated values
- [ ] Verified alignment for every primary GRE sense
- [ ] Verified reusable example for every queueable sense
- [x] Exact-sense editorial confusable graph for all 292 quiz-target senses,
      backed by a 296-sense formal option pool with four support-only senses
- [ ] Licensed/private-local workflow for ETS references and screen dialogue

## v2.2 alpha — Private question context and flexible workload · current

- [x] Private structural index for locally held fill-in and reading materials
- [x] Exact word-form index plus written human sense decisions for every
      question match selected for display
- [x] Personal question display with PDF page/locator, answer, and explicit
      distinction between confirmed sense and word-form-only evidence
- [x] Public-build stripping and serialized private-question leak checks
- [x] Editable 1–200 daily new-word target, completion forecast, and unlimited
      additional 70/30 new-word batches
- [x] Local screenshot-first mock-test mistake inbox, structured analysis, and
      spaced reinforcement
- [ ] Dedicated synonym, antonym, confusable, and intensity exercises
- [ ] Cloze and sentence-equivalence transfer activities

## v2.3 — Stronger adaptation

- FSRS-compatible memory parameters
- Interference-aware scheduling between near-synonyms
- Weekly weakness report and workload calibration
- Ambiguous legacy-sense migration review

## v3.0 — Optional sync and public beta

- Optional encrypted sync separate from the local-first core
- Accessibility audit and cross-browser migration tools
- Contributor-ready catalog and activity interfaces
